HTML Element

Description

The HTML element is used to create a website using HTML and to let system know that you are planning on using HTML.

Syntax

<html>

Example

When you create a new website, you use the HTML element to create the website using HTML.

Rendering

<html> Content </html>

Headers Element

Description

The headers element is used to create styles in the HTML code.

Syntax

<header>

Example

The headers element is an introductionary section of the HTML code, often having the style and color coding in it.

Title Element

Description

The title element creates a text in the tab for the website.

Syntax

<title>, <title>

Example

The title element is used to create the name of the page and what will be seen when browsing from a search engine like google.

Rendering

Look at the name of this tab, that's a title

Body Element

Description

The body element is used to create content onto the website itself.

Syntax

<body>, </body>

Example

Everthing you see on a website is coded inside the body element to make it appear onto the website.

Rendering

Everything you see on this website is coded inside a body element.

Headings Element

Description

The headings element is used to create large text or a paragraphs.

Syntax

Example

The headings element is often used to create titles in a website.

Rendering

Hello World!
Hello Smaller World!

Description

Syntax

Example

<hr> The paragraph element is mostly used to create simple text

Description

Description

The lines element creates horizontal lines.

Syntax

Example

Lines are used to create visual seperations between two items. Lines are a great tool to keep the website looking organized

Comments Element

Description

The comments element allows a set of text in the code itself to be discluded. So whatever is written in the comment element will be ignored and not appear in on the website.

Syntax

<!-- -->

Example

Commenting is used when you want to quickly know what a set of code is about. That way you can efficiently fix or edit the code.

Rendering
