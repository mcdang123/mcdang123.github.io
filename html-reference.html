HTML Element

Description

The HTML element is used to create a website using HTML and to let system know that you are planning on using HTML.

Syntax

<html>

Example

When you create a new website, you use the HTML element to create the website using HTML.

Rendering

Breaks Element

Description

The breaks element is used to create a new line in a paragraph.

Example

Rendering

My Bonnie lies over the sea.
My Bonnie lies over the ocean.
